# 💌 Daily Condition Letter

AI 기반 개인 맞춤형 아침 편지 시스템

## 📖 개요

매일 밤 작성한 일기와 좋아하는 문장을 바탕으로, 다음날 아침 7시에 AI가 생성한 따뜻하고 감성적인 편지를 이메일로 받아보세요.

## ✨ 주요 기능

- 📔 **일기 작성**: 매일 밤 하루를 기록
- ✨ **좋아하는 문장 저장**: 언제든 마음에 드는 문구 추가
- 🤖 **AI 편지 생성**: Google Gemini가 개인화된 편지 작성
- 📧 **자동 이메일 발송**: 매일 아침 7시 정각 (한국 시간)
- 🎨 **아름다운 템플릿**: '우리는 시를 사랑해' 뉴스레터 스타일

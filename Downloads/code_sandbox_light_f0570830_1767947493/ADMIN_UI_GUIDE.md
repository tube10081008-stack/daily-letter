# Admin UI 사용 가이드

비기술 운영자를 위한 Admin UI 완전 가이드입니다. 개발자 도구 없이 모든 일일 운영이 가능합니다.

---

## 📱 로그인

### 1. 앱 접속
```
1. 브라우저에서 배포된 URL 접속
2. 로그인 화면에서 "관리자" 탭 선택
3. ID: admin
4. PIN: 0000
5. "로그인" 버튼 클릭
```

## 🏠 Admin 홈 화면

로그인하면 표시되는 메인 화면입니다.

### 화면 구성
- **오늘 날짜**: 현재 운영 중인 날짜 표시
- **빠른 작업 버튼** (2개):
  - 🗺️ **배송지 관리**: 배송지 추가/수정/삭제
  - ➕ **오늘 배정하기**: 오늘의 배송 경로 생성
- **고급 운영 대시보드**: 실시간 모니터링 (기존 Ops 화면)
- **오늘의 기사 현황**: 각 기사별 진행 상태
  - 완료율 (%)
  - 배정된 정차지 수
  - 완료된 정차지 수
  - 메모 개수

### 화면 예시
```
┌─────────────────────────────┐
│ 운영 관리                    │
│ 2026년 1월 9일               │
├─────────────────────────────┤
│ ┌───────────┐ ┌───────────┐ │
│ │  배송지   │ │   오늘    │ │
│ │   관리    │ │  배정하기 │ │
│ └───────────┘ └───────────┘ │
├─────────────────────────────┤
│ 오늘의 기사 현황             │
│ ┌─────────────────────────┐ │
│ │ 김철수 (북부권)    85%  │ │
│ │ 배정: 4  완료: 3  메모:1│ │
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

## 🗺️ 배송지 관리

### 1. 배송지 목록 보기

#### 검색 기능
- 상단 검색창에 이름 또는 주소 입력
- 실시간으로 목록 필터링

#### 구역 필터
- **전체**: 모든 배송지 표시
- **북부**: 북부권 배송지만 표시
- **남부**: 남부권 배송지만 표시

#### 배송지 카드
각 배송지는 카드 형태로 표시됩니다:
```
┌─────────────────────────────┐
│ 강남 오피스텔        [북부] │
│ 서울시 강남구 테헤란로 123  │
│ 1층 로비 경비실에서...      │
├─────────────────────────────┤
│ [수정] [복제] [삭제]         │
└─────────────────────────────┘
```

### 2. 새 배송지 추가

#### 방법 1: 직접 추가
```
1. 우측 하단 파란색 (+) 버튼 클릭
2. 배송지 정보 입력
3. "배송지 추가" 버튼 클릭
```

#### 방법 2: 기존 배송지 복제
```
1. 복제할 배송지의 "복제" 버튼 클릭
2. 자동으로 입력된 정보 확인
3. 필요한 부분만 수정
4. "배송지 추가" 버튼 클릭
```

### 3. 배송지 정보 입력

#### 필수 항목 (⭐)
1. **배송지 이름** ⭐
   - 예: 강남 오피스텔, 서초 아파트
   - 기사가 쉽게 식별할 수 있는 이름

2. **주소** ⭐
   - 전체 주소 입력
   - 예: 서울시 강남구 테헤란로 123

3. **구역** ⭐
   - 북부권 또는 남부권 선택
   - 북부권: 강남, 서초, 역삼, 논현 등
   - 남부권: 영등포, 구로, 관악, 동작, 금천 등

4. **출입 안내** ⭐
   - 기사가 현장에서 참고할 상세 정보
   - **중요**: 자세할수록 좋습니다
   - 예시:
     ```
     1층 로비 경비실에서 방문증 수령 후
     3동 엘리베이터 이용
     1502호 비밀번호 #1234*
     ```

#### 선택 항목
5. **연락처** (선택)
   - 긴급 연락이 필요한 경우
   - 형식: 010-1234-5678

6. **태그** (선택)
   - 특성 표시를 위한 태그
   - 체크박스로 선택:
     - ☐ 경비실 방문 필요
     - ☐ 주차 어려움
     - ☐ 비밀번호 필요
     - ☐ 엘리베이터 없음

### 4. 시스템 자동 생성

**위치 ID는 시스템이 자동으로 생성합니다!**

- 북부권: `LOC-N-0001`, `LOC-N-0002`, ...
- 남부권: `LOC-S-0001`, `LOC-S-0002`, ...
- 관리자는 ID를 직접 입력하지 않습니다

### 5. 배송지 수정

```
1. 수정할 배송지의 "수정" 버튼 클릭
2. 정보 수정
3. "수정 완료" 버튼 클릭
```

**참고**: 위치 ID는 수정할 수 없습니다 (시스템 관리)

### 6. 배송지 삭제

```
1. 삭제할 배송지의 "삭제" 버튼 클릭
2. 첫 번째 확인: "삭제" 클릭
3. 두 번째 확인: "네, 삭제합니다" 클릭
4. 삭제 완료
```

**⚠️ 주의**: 삭제는 되돌릴 수 없습니다!

## ➕ 오늘 배정하기

매일 배송 경로를 생성하는 3단계 프로세스입니다.

### 단계 1: 배송지 선택

#### 1.1 날짜 선택
- 기본값: 오늘
- 변경 가능: 미래 날짜 선택 가능

#### 1.2 구역 선택
- **북부권**: 김철수 기사에게 자동 배정
- **남부권**: 이영희 기사에게 자동 배정

#### 1.3 배송지 선택
- 체크박스로 원하는 배송지 선택
- **전체 선택** 버튼: 해당 구역 모든 배송지 선택
- 최소 1개 이상 선택 필요

#### 화면 예시
```
┌─────────────────────────────┐
│ 배정 생성                    │
│ ●─────●───────○ (1/3)       │
├─────────────────────────────┤
│ 배송 날짜: 2026-01-09       │
│ 구역: ◉ 북부권  ○ 남부권    │
├─────────────────────────────┤
│ ✓ 김철수 기사에게 배정됩니다 │
├─────────────────────────────┤
│ 배송지 선택 (3개)   [전체선택]│
│ ☑ 강남 오피스텔              │
│   서울시 강남구...           │
│ ☑ 서초 아파트                │
│   서울시 서초구...           │
│ ☑ 역삼 빌라                  │
│   서울시 강남구...           │
├─────────────────────────────┤
│ [다음: 순서 정렬 (3개)]      │
└─────────────────────────────┘
```

### 단계 2: 순서 정렬

#### 2.1 자동 정렬
- 시스템이 배송지 이름 순으로 자동 정렬
- 기사가 쉽게 찾을 수 있도록 배치

#### 2.2 수동 조정
- 각 배송지 우측의 ↑ ↓ 버튼 클릭
- 원하는 순서로 재배치

#### 조정 팁
```
✓ 같은 건물/단지는 연속으로 배치
✓ 이동 동선을 고려하여 정렬
✓ 중요 고객은 앞쪽에 배치
✓ 주차가 어려운 곳은 한가한 시간대로
```

#### 화면 예시
```
┌─────────────────────────────┐
│ 순서 정렬                    │
│ ●─────●───────○ (2/3)       │
├─────────────────────────────┤
│ 버튼으로 순서를 조정하세요   │
├─────────────────────────────┤
│ 1 강남 오피스텔        [↑][↓]│
│   서울시 강남구...           │
│ 2 서초 아파트          [↑][↓]│
│   서울시 서초구...           │
│ 3 역삼 빌라            [↑][↓]│
│   서울시 강남구...           │
├─────────────────────────────┤
│ [다음: 확인 및 저장]         │
└─────────────────────────────┘
```

### 단계 3: 확인 및 저장

#### 3.1 배정 정보 확인
- 날짜
- 구역
- 기사 이름
- 정차지 개수

#### 3.2 배송 순서 최종 확인
- 1번부터 순서대로 표시
- 이름과 주소 확인

#### 3.3 알림 설정
- ☑ **기사에게 알림 전송** (기본 체크)
  - 체크 시: 배정 완료 SMS/Email 자동 발송
  - 해제 시: 알림 없이 배정만 생성

#### 3.4 저장
```
1. 정보 최종 확인
2. "저장 및 배정 완료" 버튼 클릭
3. 생성 완료 메시지 확인
4. 자동으로 홈으로 이동
```

#### 화면 예시
```
┌─────────────────────────────┐
│ 배정 확인                    │
│ ●─────●───────● (3/3)       │
├─────────────────────────────┤
│ 배정 정보                    │
│ 날짜: 2026-01-09            │
│ 구역: 북부권                 │
│ 기사: 김철수                 │
│ 정차지: 3개                  │
├─────────────────────────────┤
│ 배송 순서                    │
│ 1. 강남 오피스텔             │
│ 2. 서초 아파트               │
│ 3. 역삼 빌라                 │
├─────────────────────────────┤
│ ☑ 기사에게 알림 전송         │
├─────────────────────────────┤
│ [저장 및 배정 완료]          │
└─────────────────────────────┘
```

## ❓ 자주 묻는 질문

### Q1: 배송지 ID는 어떻게 정하나요?
**A**: 시스템이 자동으로 생성합니다. 관리자는 ID를 입력하지 않습니다.
- 북부권: LOC-N-0001, LOC-N-0002, ...
- 남부권: LOC-S-0001, LOC-S-0002, ...

### Q2: 같은 날짜에 배정을 여러 번 생성할 수 있나요?
**A**: 아니요. 같은 날짜/구역은 하나의 배정만 가능합니다.
이미 배정이 있다면 고급 대시보드에서 순서를 재정렬하세요.

### Q3: 배송지를 삭제하면 어떻게 되나요?
**A**: 완전히 삭제됩니다. 되돌릴 수 없으므로 신중하게 결정하세요.
과거 배정에 포함된 배송지도 삭제할 수 있지만, 기록은 남아있습니다.

### Q4: 기사를 추가하고 싶어요.
**A**: 현재 Admin UI에서는 불가능합니다.
DEPLOYMENT_GUIDE.md를 참고하여 개발자 도구로 추가하거나,
기술 지원팀에 요청하세요.

### Q5: 배정 생성 중 실수했어요.
**A**: 
- 단계 1-2: 뒤로가기 버튼으로 이전 단계 이동 가능
- 단계 3 저장 전: 뒤로가기로 수정 가능
- 저장 후: 고급 대시보드에서 순서 재정렬 또는 삭제 후 재생성

### Q6: 배송지가 너무 많아요. 어떻게 관리하나요?
**A**: 
- 검색 기능 활용: 이름 또는 주소로 검색
- 구역 필터 활용: 북부/남부 구분
- 태그 활용: 특성별로 분류

### Q7: 출입 안내를 어떻게 작성하나요?
**A**: 기사가 현장에서 쉽게 따라할 수 있도록 단계별로 작성하세요.
```
좋은 예시:
1층 로비 경비실에서 방문증 수령
3동 엘리베이터 이용
1502호 비밀번호 #1234*

나쁜 예시:
경비실 가서 물어보세요
```

### Q8: 기사가 배정을 못 받았다고 해요.
**A**: 
1. 배정 생성 시 "기사에게 알림 전송" 체크 확인
2. 기사에게 앱 새로고침 요청
3. 고급 대시보드 → "메시지" 기능으로 직접 알림

### Q9: 배송 순서를 바꾸고 싶어요.
**A**: 두 가지 방법:
1. 배정 생성 시: 단계 2에서 ↑↓ 버튼으로 조정
2. 배정 생성 후: 고급 대시보드 → Driver 상세 → 순서 재정렬

### Q10: 모바일에서도 사용 가능한가요?
**A**: 네! 모든 기능이 모바일 최적화되어 있습니다.
스마트폰 브라우저에서 접속하세요.

## 🆘 문제 해결

### 문제 1: 로그인이 안 돼요
```
1. ID와 PIN 확인 (admin / 0000)
2. 브라우저 새로고침 (F5)
3. 시크릿/프라이빗 모드로 재시도
4. 다른 브라우저로 시도
```

### 문제 2: 배송지 목록이 안 보여요
```
1. 검색창 확인 (검색어 입력되어 있는지)
2. 구역 필터 확인 (전체 선택)
3. 페이지 새로고침
```

### 문제 3: 배정 생성이 실패해요
```
1. 같은 날짜/구역에 이미 배정이 있는지 확인
2. 배송지를 최소 1개 이상 선택했는지 확인
3. 네트워크 연결 확인
4. 잠시 후 재시도
```

### 문제 4: 저장 버튼을 눌렀는데 반응이 없어요
```
1. 필수 항목(⭐) 모두 입력했는지 확인
2. 3-5초 기다리기 (네트워크 지연)
3. 페이지 새로고침 후 재시도
```

### 문제 5: 삭제가 안 돼요
```
1. 두 번의 확인 단계 모두 진행했는지 확인
2. 네트워크 연결 확인
3. 잠시 후 재시도
```
